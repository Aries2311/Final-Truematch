# TrueMatch Local Run (Windows)

## 1) Install
npm install

## 2) Create .env
Copy `.env.example` -> `.env` and fill real values.

## 3) Firebase Admin service account
Place your real `firebase-service-account.json` inside:
`backend/firebase-service-account.json`

(Do NOT commit it)

## 4) Run
npm start

Open: http://localhost:3000
